# Go Home Estimator

### Definisi Tugas
Anda diminta untuk membuat aplikasi “Go Home Estimator”, sebuah one-pager (satu halaman web) yang menampilkan waktu yang dibutuhkan untuk menyetir pulang dari kantor.
